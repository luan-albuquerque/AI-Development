# 📊 Previsão de Receita de Vendedores com Regressão

Neste desafio, você atuará como **analista de dados** em uma empresa de vendas, com a missão de prever a receita gerada por vendedores a partir de informações como tempo de experiência, número de vendas e fatores sazonais.

O objetivo é aplicar técnicas de regressão para analisar o impacto dessas variáveis e identificar o modelo mais eficiente para previsão de receita.

## 🗂️ Dataset

O dataset contém as seguintes variáveis:

- `tempo_de_experiencia`: Tempo em meses que o vendedor trabalha na empresa.  
- `numero_de_vendas`: Número de vendas realizadas pelo vendedor em um período específico.  
- `fator_sazonal`: Fator de 1 a 10 que representa a sazonalidade das vendas (ex: 10 representa alta temporada).  
- `receita_em_reais`: Receita total gerada pelo vendedor.  

---

## ✅ Tarefas a serem feitas

- [ ] Carregue o dataset `sales_data.csv` utilizando o Pandas.  
- [ ] Realize uma análise exploratória dos dados, identificando correlações entre as variáveis.  
- [ ] Separe os dados em conjuntos de treinamento e teste.  
- [ ] Treine um modelo de regressão linear para prever a receita com base no tempo de experiência e no número de vendas.  
- [ ] Avalie o desempenho do modelo linear utilizando métricas como MSE e R².  
- [ ] Treine um modelo de regressão polinomial, incluindo o fator sazonal como variável adicional.  
- [ ] Avalie o desempenho do modelo polinomial e compare com o modelo linear.  
- [ ] Compare os resultados e decida qual modelo é mais adequado para prever a receita com base nas variáveis fornecidas.
